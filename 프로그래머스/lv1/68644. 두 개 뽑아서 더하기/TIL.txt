vector에서 정렬 & 중복제거 동시에 하고싶을때


vector<int> v; // v == {0, 2, 2, 1, 1, 1, 3} 이라고 해보자

sort(v.begin(), v.end()); // v == {0, 1, 1, 1, 2, 2, 3}
v.erase(unique(v.begin(), v.end()), v.end());
// unique 이후에는 v == {0, 1, 2, 3, 1, 1, 2}
// erase 이후에는 v == {0, 1, 2, 3}

위와 같이 작성하면
일단 6번째 줄에서 정렬이 될 것이고
7번째 줄의 unique에서 중복되는 쓰레기 값들을 뒤로 보낸 후, 첫 번째 쓰레기 값의 위치를 반환한다
따라서 erase 함수로 쓰레기 값들을 지워주면 정렬 -> 중복제거 를 할 수 있다.
